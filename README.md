### Featured work

**FreqShield-ViT** — Frequency-domain adversarial defenses for Vision Transformers
- *Stack*: PyTorch · DeiT-Small · torch-dct · PyWavelets · SLURM
- *Why*: Standard adversarial training degrades clean accuracy. Spectral analysis suggests low-frequency components carry most of the perturbation budget.
- *Impact*: Reproducible pipeline (train / eval / ablations / patch-attack), full spectral diagnostics, paper in draft.

**sp500-tft-forecasting** — Temporal Fusion Transformers for S&P 500 returns
- *Stack*: PyTorch Lightning · pytorch-forecasting · ARIMAX / LSTM baselines
- *Why*: Compare attention-based sequence models to classical baselines on noisy financial data, with honest failure analysis.
- *Impact*: 450+ runs across 11 experiment phases; gradient-collapse diagnostics and limitations documented.

**BasketIQ** — Recommendation foundations on 32.4M Instacart transactions
- *Stack*: Python · mlxtend (Apriori) · scikit-learn · Chart.js
- *Why*: Association-rule mining + RFM segmentation as the classical scaffold underneath modern recommendation systems.
- *Impact*: Top-50 product association rules, K-Means customer segments, interactive dashboard.

**Trait-BERT** — Big Five personality from LinkedIn text *(repo: Analysing-Personality-from-LinkedIn-Profile)*
- *Stack*: HuggingFace Transformers · BERT fine-tuning · scikit-learn
- *Why*: Domain-specific transformer fine-tuning for multi-label text classification — same primitive as intent detection and structured extraction.
- *Impact*: Multi-author collaboration (Traitlytics), comparative baselines documented.
